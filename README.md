## Student Result Management System

Objective: To integrate all the learned concepts into a working application using HTML, CSS, JS and that can perform Create, Read, Update, Delete operations using Ajax and JSON Server.

### Project Requirements:
• Define classes such as Student, Sections, Result.
• Build an UI using HTML, CSS.
• Use JS Async functions to perform CRUD operations
• Handle HTTP request and responses gracefully
• Store and retrieve student data from JSON Server and display the data in the UI.

### Expected Learning Outcomes:
• Apply HTML, CSS, JS principles in real-world context.
• Implement clean, reusable, and maintainable code in HTML, CSS, JS.
• Perform CRUD operations using JSON Server and display the data in the UI.
